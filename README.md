# network-graph

Minimal standalone repo for Q2 from the homework.

Contents:
- `Q2/Q2.html`
- `Q2/board_games.csv`
- `lib/d3.v5.min.js`
- `index.html`

Run locally with:

```powershell
python -m http.server 8000
```

Then open:

`http://localhost:8000/Q2/Q2.html`

GitHub Pages:

- Site root: `/`
- Graph page: `/Q2/Q2.html`
